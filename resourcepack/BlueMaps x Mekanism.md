**Stop players from scouting ores through your web map!**

By default, BlueMap renders blocks exactly as they appear, which can lead to players using the web interface to pinpoint rare Mekanism ores. This resource pack provides a simple, lightweight fix by overriding the model files for all Mekanism ores, making them **completely invisible** specifically within the BlueMap render.

### ✨ Key Features

* **Anti-Cheat:** Prevents players from finding Osmium, Tin, Lead, Uranium, and Fluorite via the web browser.
* **Performance Friendly:** Uses simple model overrides no complex scripts or heavy textures.
* **Seamless Integration:** Does not affect in-game visuals; ores remain perfectly visible to players while they are actually playing the game.

### 🛠 Installation Guide

To apply these changes to your map, follow these steps:

1. **Download** the latest version of the resource pack.

2. **Upload** the `.zip` file to your server's BlueMap configuration directory: Path: `config/bluemap/packs/` (or the equivalent for your mod loader).

3. **Refresh Renders:** For the changes to take effect on existing chunks, you must perform a full re-render: `/bluemap reload`


4. **Enjoy!** Your Mekanism ores are now hidden from prying eyes on the web.

---

> **Note:** This pack specifically targets Mekanism ores. If you are using other tech mods, you will need another pack that covers those, but you can open a suggestion on our GitHub and we can upload one specifically for that too!
